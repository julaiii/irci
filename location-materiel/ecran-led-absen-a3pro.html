Écran LED Absen A3 Pro

Année : 2024 Catégorie : Vidéo

Description

Écran LED haute résolution pour événements indoor

L'**Absen A3 Pro** est un écran LED modulaire haute résolution conçu pour les événements indoor professionnels. Avec un **pitch de 3mm**, il offre une **qualité d'image exceptionnelle** même à courte distance. Parfait pour les conférences, salons, concerts, galas et événements corporate nécessitant une diffusion vidéo en grand format avec une image ultra-nette.
